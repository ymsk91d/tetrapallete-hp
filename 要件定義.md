## 0. このドキュメントの位置づけ

既存サイト（Canva製）・各グループのX・YouTube・配信状況を調査した上で、
新しいコーポレートサイトの**目的・全体像・情報設計・デザイン方針・機能要件**を定義する。
本書は次工程「画像生成AIによるデザインカンプ作成」への発注材料も兼ねる（→ §9 デザイン要件）。

## 6. サイトの目的・成果指標（案）

- **KGI**: ①オーディション応募数 ②映像/イベント制作の問い合わせ数 ③グループ認知・ファン獲得
- **KPI例**: LINE応募遷移率 / 問い合わせ送信数 / YouTube・SNS外部遷移数 / MV再生 / 滞在時間 / モバイル離脱率
- **既存サイトの課題（改善対象）**:
  - 独自ドメインでない（`*.canva.site`）→ ブランド・SEO・信頼性が弱い
  - 動画が実埋め込みでない（Canva内部アセット）→ 視聴体験・回遊が弱い
  - 映像事業（BtoB）の受け皿がない
  - 表示の重さ・遅延読み込み、構造化データ・OGP・サイトマップ等のSEO基盤が不足
  - スマホ最適化の作り込み余地

## 7. 情報設計（サイトマップ案）

トップを「会社の全体像のハブ」とし、3軸へ分岐。**マルチページ構成**を推奨。

```
TOP（会社の世界観・全体像／各セクションへのハブ）
├─ ABOUT / COMPANY      会社概要・理念・沿革・代表メッセージ・事業領域
├─ ARTISTS（TALENT）     所属グループ一覧 → 各グループ詳細 → メンバープロフィール
│   ├─ HALO PALLETE
│   ├─ ヒトノユメ
│   └─ Frolic Park
├─ MOVIE / WORKS        ★新設：映像事業（制作実績・できること・依頼の流れ・問い合わせ）
│                         └ MV / 歌ってみた / 企画 / Dance Practice をカテゴリ表示・動画埋め込み
├─ AUDITION             オーディション（提供・応募資格・審査フロー・FAQ・LINE応募）
├─ NEWS / LIVE          ライブ・リリース・お知らせ（任意：TimeTree連携 or 手動）
└─ CONTACT              用途別問い合わせ（映像/イベント発注・取材・その他）＋会社情報
                          フッター：全SNS・LINE・所在地・各種ポリシー
```

- グローバルナビ案: `ABOUT / ARTISTS / MOVIE / AUDITION / CONTACT`（＋LINE・SNSアイコン）

## 8. 機能要件

### 8-1. 動画埋め込み（最重要・映像事業の核）
- **YouTube（公式チャンネルの実埋め込み）** を標準採用（`youtube-nocookie`／遅延読込でパフォーマンス確保）
- カテゴリ別ギャラリー（MV / 歌ってみた / 企画 / Dance Practice）。**カードのサムネ → クリックで再生**（初期はサムネ画像のみ読込＝軽量化）
- 1本を大きく見せる**ヒーロー動画枠**（映像事業ページ／トップに「制作実績ショーケース」）
- **将来の追加が容易なデータ構造**（動画リストを配列/JSONで管理し、URL追記だけで増やせる）
- 自社撮影のMP4を置きたい場合に備え、`<video>`直埋めも選択肢として用意

### 8-2. 映像事業ページ（新設）
- ヒーロー（代表作MVを大きく）／提供サービス（MV・PV・YouTube企画・ライブ撮影・編集 等）
- 制作実績ギャラリー（動画グリッド）／制作の流れ／料金やプランは出す/出さないを要確認
- **CTA: 映像制作のお問い合わせ**（フォーム or メール/LINE）

### 8-3. その他
- 各グループ・各メンバーの**SNSリンク集約**（X / Instagram / TikTok / YouTube）
- **LINEオーディション応募**ボタン（`lin.ee/KdHDhzV`）を全体で目立たせる
- 問い合わせ：用途別（映像/イベント/取材/その他）。送信手段は要決定（フォーム送信 or mailto/LINE）
- レスポンシブ（スマホ最優先）／OGP・favicon・サイトマップ・構造化データ（Organization/VideoObject/FAQ等）

## 9. デザイン要件（→ 画像生成AIへの発注材料）

### ブランドの軸
- **二層構造**:「**コーポレートの落ち着き**（信頼・大人・映像事業の説得力）」×「**アイドルのカラフルさ**（青春・多色・高揚）」
- 既存ロゴカラー: **オレンジ/アンバー × ダークグレー/チャコール**
- キーワード: 想い・夢・未来 ／ カラフル ／ 王道 ／ 青春 ／ ステージの熱量

### トーン&マナー（カンプ生成プロンプト用メモ）
- ベースはダークまたはニュートラルの上質なコーポレート基調。差し色にオレンジ＋各グループ担当カラー（白・水色・ミント・ピンク・オレンジ・紫・黄 等のカラフル）
- ヒーローはライブ写真／MVスチル（実際に紙吹雪・照明・ステージの熱量がある素材あり）
- 見出しは明朝（既存の「想いを重ねて〜」が上品な明朝）＋本文ゴシックの和欧混植
- 映像事業セクションは「制作会社らしい」シネマティックな質感（横長16:9の動画グリッド、黒地に映える）
- 全体に余白を効かせ、スマホでの可読性・タップしやすさを最優先

### 画像生成AIに渡す想定（このあと別途プロンプト化して提供）
- TOP / ARTISTS一覧 / グループ詳細 / MOVIE（映像事業）/ AUDITION の各カンプ（PC・SP）
- ロゴ・担当カラー・上記トンマナを指定

## 10. 技術要件（案）

- **静的サイト（HTML / CSS / 最小限のJS）** で手組み（高速・軽量・保守容易）。フレームワーク不要。
- 独自ドメイン取得・常時HTTPS・適切なホスティング（要決定）
- パフォーマンス: 画像最適化（WebP）、動画はサムネ→クリック再生、遅延読込
- SEO: メタ/OGP、`sitemap.xml`、`robots.txt`、JSON-LD（Organization・VideoObject・FAQPage・BreadcrumbList）、ローカルSEO
- アクセシビリティ: 代替テキスト・コントラスト・キーボード操作
- 解析: アクセス計測（要決定：GA4 等）
