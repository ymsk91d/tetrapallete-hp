TetraPallete — 事業ビジュアル画像の差し込み場所
================================================

このフォルダに以下のファイル名で画像を置くと、サイトに自動で反映されます。
（未配置の間は、各事業色のグラデーション＋ラベルのプレースホルダが表示されます）

ファイル名（すべて .jpg）:
  talent.jpg  … タレント事業（TOPハブカード ＋ talent.html ヒーロー帯）
  video.jpg   … 映像事業（TOPハブカード ＋ video.html ヒーロー帯）
  music.jpg   … 音楽事業（TOPハブカード ＋ music.html ヒーロー帯）
  studio.jpg  … 貸しスタジオ事業（TOPハブカード ＋ studio.html ヒーロー帯）
  about.jpg   … 会社概要セクションの帯（index.html ABOUT）

推奨スペック:
  - 横長・2000×1000px 前後（2:1）。JPG、長辺2000px・500KB〜1.5MB目安。
  - 同じ画像をカード(16:10)とヒーロー帯(21:8)の両方で中央クロップして使うため、
    主役は中央寄せ・上下左右に余白を残す構図が安全。
  - トーンは「明るく上質・ミニマル」。各事業のシグネチャ色を“差し色”に。
    （talent=ピンク / video=パープル / music=オレンジ / studio=ミント）

差し替え後はブラウザを再読み込みするだけで表示されます。
