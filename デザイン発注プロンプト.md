# デザインカンプ 画像生成AI 発注プロンプト集

目的: 画像生成AIにWebサイトのデザインカンプ（画面イメージ）を描いてもらうためのプロンプト。
方向性を決めるため **①コーポレート寄り／②アイドル寄り** の2案を用意。まず各案の **TOP(PC)** を比較し、方向を決定 →
決まった案で残り画面（ARTISTS / AUDITION / CONTACT）を追加発注する。

---

## 使い方・共通の注意

- 画像AIは**文字（特に日本語）をきれいに描けない**ため、文字化けは気にしない。**レイアウト・配色・雰囲気**を見るのが目的。実際の文言はコード実装時にこちらで正しく入れる。
- 各プロンプトの先頭に下の【共通設定】を付けて投げると統一感が出る。
- 比率は **PC=16:9（横長）／スマホ=9:19（縦長）** を指定。
- ツールが英語の方が得意な場合は「英訳して」と言ってください（別途英語版を出します）。

### 【共通設定】（毎回先頭に貼る）

```
ブランド: 「TetraPallete（テトラパレット）」＝アイドルグループをプロデュースし、MV等の映像制作も手がける会社。
ロゴ: オレンジ色の手書き風スクリプトで "Tetra Pallete" ＋ 円形フレーム。"HP" のモノグラム（オレンジのグラデーション円）。
コーポレートカラー: オレンジ／アンバー（#FF8A3D〜#F5A623）× チャコール／ダークグレー（#1A1A1A〜#2B2B2B）× オフホワイト。
所属3グループの担当カラー（差し色に使用可）: ホワイト・水色・ミントグリーン・ブロッサムピンク・オレンジ・パープル・イエロー。
書体: 見出しは上品な明朝体、本文はゴシック体。和文中心。
出力: 実在するWebサイトのデザインカンプ（フルページのスクリーンショット風）。高精細、プロのWebデザイン、整然としたグリッド。
文字は読めなくてよい（プレースホルダー扱い）。レイアウトと配色と世界観を重視。
```

# ◆ 採用テイスト確定（案②アイドル寄りベース）＋ 差別化・素材生成プロンプト

採用カンプ: 案②アイドル寄り（カラフル／パステル虹／紙吹雪／青春ポップ）。これを **基準テイスト** とする。
ゴール: 「よくあるアイドル/事務所サイト」から**配置（レイアウト）と動き（アニメーション）で差別化**する。
そのために、平面カンプを切るだけでなく **透過の個別パーツ（人物・紙吹雪・キラキラ・ブロブ・装飾・アイコン）** を用意し、
レイヤーで重ねてCSS/JSでアニメさせる。

## 基準テイスト（taste lock：全プロンプト共通の世界観）

```
テイスト: 明るくカラフルな青春ポップ。白〜淡いパステルの虹色グラデーション基調、オレンジが主役の差し色。
紙吹雪（カラフルな短冊状）、やわらかい光のbokeh、キラキラ/星のスパークル。手書き風のあしらい。
かわいい×かっこいいの王道アイドル感。丸み・グラデ・透明感。高精細でプロのクオリティ。
配色: オレンジ#FF8A3D〜#F5A623 / パステル(ピンク#FFB3D9・水色#A9E0FF・ミント#B8F2D8・パープル#D9B3FF・イエロー#FFE89E) / オフホワイト#FFFDF7。
```

## A. 差別化コンセプト（配置・動き）※実装でClaudeが担保

**配置（既存との差別化）**
- 整列グリッドをやめ、**非対称・斜め分割・レイヤー重なり**。写真がセクション境界や文字を**はみ出す**。
- ヒーローは中央寄せの定番をやめ、**人物を左右に大きく振り、中央に縦組み/斜めの大見出し**。
- ARTISTS は横並びカードでなく、**横スクロールの帯**や**ジグザグ配置**で“雑誌のような”見せ方。
- セクション境界に**手書き風の波線/紙テープ**を使い、四角い箱の連続を避ける。

**動き（既存との差別化）**
- 背景に**紙吹雪・キラキラの常時パーティクル**（canvas）。スクロール速度・カーソルに反応。
- スクロール連動で**人物がスライドイン／フェードイン**、見出しが**1文字ずつ出る**（kinetic typography）。
- ヒーロー写真の**ゆっくりズーム/パララックス**、パステルの**ブロブがゆらゆら**変形。
- 数字（楽曲数・MV数・ライブ本数）の**カウントアップ**。
- ホバーで**浮く／傾く**カード、**カスタムカーソル**にきらめき追従。
- ページ読み込み時の**ロゴ→紙吹雪が舞うオープニング演出**。
- ※やり過ぎ防止: 動きは“軽快に短く”、スマホは負荷を抑えて簡略化。

## B. 素材生成プロンプト（画像AI用・★背景透過の個別パーツ）

> 透過PNG（アルファ）対応のツールで。1パーツ＝1生成。アニメ・自由配置のために**単体**で作るのが肝。

**B-1 背景（シームレス）**
```
【基準テイスト】を適用。Webサイトの背景画像。淡いパステルレインボーのなめらかなグラデーション、
ふんわりした光のbokeh、ごく薄く紙吹雪が散る。文字や人物は無し。横長・高精細・シームレスでタイル可能。主張しすぎない淡さ。
```

**B-2 紙吹雪パーツ一式（透過）**
```
【基準テイスト】を適用。背景透過(PNG)。色とりどりの紙吹雪（短冊・正方形・丸・リボン状）が
ばらばらに散ったセット。オレンジ/ピンク/水色/ミント/パープル/イエロー。影付きで立体感。
個別に切り離して使えるよう余白に散らす。人物・文字なし。
```

**B-3 キラキラ／スパークル（透過）**
```
【基準テイスト】を適用。背景透過(PNG)。きらめき・星・4点スパークル・小さな光の粒のセット。
白〜パステル、オレンジのアクセント。大小さまざま。人物・文字・背景なし。
```

**B-4 パステルのブロブ／有機的シェイプ（透過）**
```
【基準テイスト】を適用。背景透過(PNG)。やわらかい有機的なブロブ（雲のような丸い形）を数種、
パステル虹色のグラデで。にじみ・透明感。装飾の土台に使う。文字なし。
```

**B-5 手書き風あしらい（透過）**
```
【基準テイスト】を適用。背景透過(PNG)。手書き風の装飾セット：丸囲み・下線・吹き出し・矢印・
星・ハート・音符・キラキラ。オレンジ＆カラフル、明るくポップ。ラフな線。文字は入れない。
```

**B-6 事業アイコン3種（透過・統一テイスト）**
```
【基準テイスト】を適用。背景透過(PNG)。フラットでかわいいラインアイコン3つを同じテイストで：
(1)マイク=アイドルプロデュース (2)ビデオカメラ/フィルム=映像制作 (3)ステージ/ライト=イベント制作。
オレンジ〜パステルのグラデ、丸み。3つ並べて。文字なし。
```

**B-7 ロゴ（透過・作り直し用／もしくは支給ロゴを使用）**
```
【基準テイスト】を適用。背景透過(PNG)。円形フレームの中に手書き風スクリプト "Tetra Pallete"、
右上に "HP" のオレンジ円形モノグラム。オレンジのグラデ。ロゴ単体、余白に配置。
※既存の正式ロゴ画像が支給できる場合はそちらを優先（生成は仮）。
```

## C. 画像切り出し指示（生成カンプ／支給写真から Claude が抽出）

> 「動かす・自由に置く」ために、人物は**背景を消して透過で切り抜く**のが前提。

- **メンバー集合写真**: 人物グループだけを**透過切り抜き**（背景・紙吹雪を除去）。左右パートに分けられるとヒーローで振り分け可能。
- **各グループ写真（3点）**: グループごとに人物を透過切り抜き。ARTISTSカード／詳細用。
- **ロゴ**: 透過で抽出（または支給の正式データ）。ヘッダー/フッター/オープニング用。
- **背景レイヤー**: 紙吹雪・光は B-1〜B-3 の生成素材に置き換え（カンプ背景は参考）。
- 切り抜きは Claude 側で実施（透過PNG化）。**元の高解像度写真の支給があれば品質が大きく上がる**ので、可能なら原本を共有。

## D. 出してほしいカンプ（このテイスト＋差別化前提）

各画面、**この基準テイストのまま**、Aの「配置の差別化」を意識して生成:
- TOP（PC / SP）※採用済みベースを、非対称・はみ出し配置で再生成しても良い
- ARTISTS（一覧＝横スクロール帯／詳細＝メンバーカード）
- MOVIE（映像事業：シネマティック寄りに少し締める）
- AUDITION / CONTACT

> 動き（A）は静止画カンプには映らないため、**実装時に Claude が付与**する。カンプは配置・配色・素材の確認用。
